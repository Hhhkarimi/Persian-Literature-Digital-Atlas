# محل فایل مجموعه‌داده

فایل واقعی مجموعه‌داده را با نام دقیق زیر در همین پوشه قرار دهید:

```text
poems_with_more_info.tsv
```

مسیر نهایی در ریشهٔ مخزن باید چنین باشد:

```text
public/data/poems_with_more_info.tsv
```

فایل TSV داخل این پکیج قرار نگرفته است. آن را از مخزن زیر دریافت کنید:

https://github.com/aghasemi/ChronologicalPersianPoetryDataset/blob/master/poems_with_more_info.tsv

چون حجم فایل حدود ۸۰ مگابایت است، بارگذاری از رابط وب GitHub ممکن نیست. از GitHub Desktop یا Git CLI استفاده کنید. مطمئن شوید خود فایل TSV را Push می‌کنید، نه صفحهٔ HTML دانلودشده و نه اشاره‌گر Git LFS.

ستون‌های مورد انتظار:

```text
poet	century	book_title	poem_title	poem
```
